This tool can be used in any Test Center, it has:
1) Desktop application developed using JAVA to retrieve the information from MSSQL database and acts in Test Mode.
2) Web Application developed using ASP.NET to add tests, questions, test takers, marking and correcting answers if needed like essay questions, communication with test takers.
